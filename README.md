# DBZ Budokai Tenkaichi 3 – Projeto de Mod

Este é um projeto pessoal de mod para DBZ Budokai Tenkaichi 3. Estou começando agora no modding, então a ideia é aprender enquanto faço. Essa ISO reúne mudanças simples e testes que vou melhorando com o tempo.

O objetivo é criar uma versão melhorada do jogo, deixando tudo mais organizado e consistente. Não é algo finalizado, e muita coisa ainda pode mudar conforme eu for entendendo melhor como o jogo funciona.

As alterações são feitas aos poucos, versão por versão. Algumas coisas podem não ficar perfeitas no começo, mas fazem parte do processo de aprendizado.

Este não é um trabalho profissional. É um projeto de iniciante, feito para estudar, testar ideias e evoluir com o tempo.

Pretendo continuar atualizando e adicionando melhorias conforme eu aprender mais sobre modding.
